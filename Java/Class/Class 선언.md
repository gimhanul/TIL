## 4-3. Class 선언


선언은 다음과 같이 함. 클래스 이름의 첫 글자는 대문자로 선언함.

```java
//클래스이름.java
public class 클래스이름 {

}
```

- 하나의 파일에는 하나의 클래스만 선언 가능한가?
    
    아님. 두 개 이상의 클래스를 선언할 수 있음. 이를 컴파일하면 byte code file(.class)는 클래스를 선언한 만큼 생김. 결국 소스 파일은 클래스 선언을 담고 있는 저장 단위일 뿐, 클래스 자체는 아님. 또한 한 파일에 여러 개의 클래스를 선언할 때에는 `public 접근 제한자`를 주의해야 함. public 접근 제한자는 파일 이름과 동일한 이름의 클래스 선언에만 붙일 수 있음.
