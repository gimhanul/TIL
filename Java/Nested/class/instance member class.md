### 7-1-2. 인스턴스 멤버 클래스

static 키워드 없이 중첩 선언된 클래스. 

<br>

**특징**

- 인스턴스 필드와 메소드만 선언이 가능함.
- 정적 필드와 메소드를 선언할 수 없음.

<br>

**생성**

바깥 클래스를 A, 내부 클래스를 B라고 했을 때 B 클래스를 생성하려면

- 외부에서는 다음과 같이 생성할 수 있음.

```java
A a = new A();
A.B b = a.new B();
b.field1 = 3;
b.method1();
```

> 👉🏻 먼저 A 객체를 생성하고 B 객체를 생성해야 함.

<br>


- 내부에서는 다음과 같이 생성할 수 있음.

```java
class A {
	class B { }
	
	void methodA() {
		B b = new B();
		b.field = 3;
		b.method1();
	}
}
```

> 👉🏻 일반 클래스처럼 B 객체를 생성할 수 있음.
