### 9-5-5. 포장 값 비교

Boxing된 값이 다음 범위의 값이라면 `==`과 `!=` 연산자로 내부의 값을 바로 비교할 수 있음.

| 타입 | 범위 |
|---|---|
| boolean | true, false |
| char | \u0000 ~ \u0071 |
| byte, short, int | -128 ~ 127 |

하지만 다음 범위에서 벗어난다면

`Unboxing` 하여 비교하거나, `equals()` 메소드를 사용하여 비교함.

> 👉🏻 포장 객체의 내부 값을 비교하기 위해 ==과 ≠ 연산자는 사용하지 않는 것이 좋음.
